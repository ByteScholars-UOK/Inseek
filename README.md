# Inseek - Online Job Searching Platform

[CSCI 12562 Web Programming](https://fct.kln.ac.lk/degree-programmes/computer-science) group project by ByteScholars-UOK.

We made a job search website with 8 pages using html, tailwind css (cdn), javascript and jquery. users can search jobs, apply with resume upload, manage their profile, save listings and browse companies.

there are 8 members in the group and each person worked on different parts.

## what we built

- index.html - home page with search bar and featured jobs (Nimeth)
- dashboard.html - user dashboard with stats and application table (Rithika)
- profile.html - candidate profile with bio, skills, experience (Nisagi)
- auth.html - sign up and login forms with js validation (Dinuga)
- categories.html - job categories grid with hover effects (Yasith)
- apply.html - application form with file upload (Prabodha)
- companies.html - company directory with embedded video (Tharush)
- saved-jobs.html - saved jobs table with date in footer (Dulina)
